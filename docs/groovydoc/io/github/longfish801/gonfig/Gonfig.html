Package: io.github.longfish801.gonfig

[Groovy] Trait Gonfig

      • io.github.longfish801.gonfig.Gonfig

      • 設定値を保持するための特性です。
        設定値はリソース上の設定ファイルから参照し、 ConfigObjectとして保持します。
        設定ファイルは ConfigSlurperで 解析できる形式で記述してください。
        リソースの取得には GropedResourceを用います。
        基底ディレクトリは設定しないため、デフォルト値である カレントディレクトリとなります。
        設定ファイルを参照するためのリソース名は、デフォルトでは クラスの単純名をすべて小文字に変換した文字列と 拡張子(.groovy)を連結した文字列を用います。
        getResourceFileName()メソッド、 getResourceFileExtension()メソッドを オーバーライドすることでリソース名を変えることができます。
        本特性の実装時は GropedResource.getClazzの オーバーライドが必要なことに注意してください。
